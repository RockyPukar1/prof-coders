# Terminal
-> Command Line Input
1) pwd -> print working directory/folder
2) ls -> list files and folders
3) cd <path> -> change directory/folder
4) find -> to list all directory recursively
5) mkdir <foldername> -> make directory
6) touch <filename> -> create file
7) rm <filename> -> remove file
8) rm -rf <folder> -> remove -recursively force
9) clear -> clear terminal
