# Data type
1. Primitive (simple)/ single value
  -> integer, floating numbers(decimal numbers), Boolean(True, False), String(double/single quote, not both)
2. Non-primitive (mixture of primitive)/ multi-value
  -> List/Array(index(number starting 0) based), Dictionary/Object(key: value pair)
  -> Tuple(same as list, but immutable), Set(Same as list, no duplicate)
